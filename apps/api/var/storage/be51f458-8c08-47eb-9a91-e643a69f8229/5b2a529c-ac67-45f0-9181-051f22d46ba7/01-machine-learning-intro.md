# Giới thiệu về Machine Learning

Machine learning là một nhánh của trí tuệ nhân tạo tập trung vào việc xây dựng
các mô hình có khả năng học từ dữ liệu mà không cần lập trình tường minh từng quy tắc.

## Các loại machine learning

- **Supervised learning**: mô hình học từ dữ liệu đã được gán nhãn.
- **Unsupervised learning**: mô hình tự tìm cấu trúc trong dữ liệu không nhãn.
- **Reinforcement learning**: mô hình học thông qua tương tác với môi trường và phần thưởng.

## Neural network và deep learning

Neural network là nền tảng của deep learning — một tập hợp các lớp (layers) xử lý
dữ liệu qua nhiều bước biến đổi phi tuyến. Deep learning đã tạo ra bước đột phá lớn
trong các bài toán như nhận dạng hình ảnh, xử lý ngôn ngữ tự nhiên, và gần đây nhất
là các mô hình ngôn ngữ lớn (LLM).

Để huấn luyện một mô hình neural network hiệu quả, cần có:
1. Dữ liệu chất lượng, đủ lớn.
2. Kiến trúc mô hình phù hợp với bài toán.
3. Tài nguyên tính toán (GPU/TPU).
4. Kỹ thuật regularization để tránh overfitting.

Deep learning framework phổ biến hiện nay gồm PyTorch, TensorFlow và JAX.
